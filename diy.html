__TECLA__

TECLA is a DIY pocket synthesizer and MIDI device
powered by a raspberry pi pico designed to expand it's users
creativity. It is 100% modifiable on both hardware and software.
Uses open source code I created in collaboration with the other
creators of this device and will be expanded to have functionalities
such as 4 digital voice polyphony and handling classic chips for
creating various retro FM sounds and experimental noise.

CircuitCircle LDeRRoR

This synth was made at mutan-lab
workshop using an electronics kit provided there.
This cool hands-on analogue stereo synth modulates
6 square oscilllators using light sensors,
applying resistance by touching or
modifying it by patching any electronics!
